Reports & Analytics API

API REST para generación de reportes en tiempo real sobre datos institucionales. Soporta filtros dinámicos, paginación, exportación a CSV y PDF, y está protegida con JWT. Construida como módulo backend complementario a los sistemas SICEL y SISA.

🔑
Autenticación requerida — Bearer JWT
Todos los endpoints (excepto /auth/login) requieren un token JWT válido en el header Authorization.
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
